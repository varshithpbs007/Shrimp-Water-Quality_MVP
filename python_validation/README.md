# WLAN Validation

This folder contains Python scripts to observe WLAN validation automation concepts relevant to embedded Wi-Fi and IoT systems.

## Scripts

### 1. WLAN KPI Log Parser and CSV report generator
Parses simulated WLAN logs and extracts:
- RSSI
- MCS index
- PHY rate
- TCP throughput
- Latency
- Retry rate

Generates a CSV report of the extracted KPIs from the test points, which can be opened in excel or as a text.
